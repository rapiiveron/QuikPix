# QuikPix v2.0 - Quick Start Guide

## üì± Installation

### Step 1: Download APK
Download `app-debug.apk` from this release.

### Step 2: Enable Unknown Sources
1. Go to **Settings** ‚Üí **Security**
2. Enable **Unknown Sources** (allow installation from unknown sources)

### Step 3: Install APK
1. Open the downloaded APK file
2. Tap **Install**
3. Wait for installation to complete

### Step 4: Grant Permissions
1. Open QuikPix for the first time
2. You'll be prompted to grant storage permission
3. Tap **Allow** or **Grant Permission**

### Step 5: Browse Photos
1. Tap on a category to view images
2. Categories are automatically detected from your device
3. Browse photos organized by folders

## üéØ Features Overview

### Category System
- **Automatic Detection** - Scans all folders on your device
- **Thumbnail Preview** - See preview images from each folder
- **Item Count** - Shows number of photos in each folder
- **Sort Options** - Sort by Recent, Name, Count, or Pinned

### Permissions
- **Android 8.0-10**: READ_EXTERNAL_STORAGE
- **Android 11-13**: READ_MEDIA_IMAGES, READ_MEDIA_VIDEO
- **Android 14+**: READ_MEDIA_VISUAL_USER_SELECTED
- **Easy Access** - Tap "Open Settings" to grant permissions

### Performance
- **Lazy Loading** - Images load as you scroll
- **Image Caching** - Previously viewed images are cached
- **Crash Protection** - Limited to 20 images per category
- **Smooth Animations** - 60fps animations

## üêõ Troubleshooting

### App Crashes
- **Issue**: App crashes when opening WhatsApp
- **Fix**: This is expected - WhatsApp has been limited to 20 images per category

### Permission Denied
- **Issue**: Can't access photos
- **Fix**:
  1. Tap "Open Settings" button
  2. Grant storage permission
  3. Restart the app

### No Categories Found
- **Issue**: No photos showing
- **Fix**:
  1. Make sure you have photos in your gallery
  2. Check that you've granted storage permission
  3. Restart the app
